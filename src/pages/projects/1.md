<h2 data-locale-en="About this project" data-locale-es="Sobre este proyecto">About this project</h2>

<p data-locale-en="Fullstack system to manage barbershop schedules." data-locale-es="Sistema fullstack para gestionar agendas de barberias.">Fullstack system to manage barbershop schedules.</p>

<p data-locale-en="The goal was to develop a backend that would handle all requests from different frontends that were on a whitelist." data-locale-es="El objetivo fue desarrollar un backend capaz de atender requests desde distintos frontends autorizados.">The goal was to develop a backend that would handle all requests from different frontends that were on a whitelist.</p>

<p data-locale-en="Additionally, each frontend would have its own associated database, allowing data isolation between clients." data-locale-es="Ademas, cada frontend tiene su propia base de datos asociada, permitiendo aislamiento de datos entre clientes.">Additionally, each frontend would have its own associated database, allowing data isolation between clients.</p>

<p data-locale-en="On the frontend side, there is a calendar that shows available time slots to book appointments at the barbershop." data-locale-es="Del lado del frontend, hay un calendario que muestra horarios disponibles para reservar citas en la barberia.">On the frontend side, there is a calendar that shows available time slots to book appointments at the barbershop.</p>

<h2 data-locale-en="Walkthrough" data-locale-es="Recorrido">Walkthrough</h2>

<p data-locale-en="In this project, I aimed to offer the system I had developed in version 1 as a service. To achieve this, I faced the challenge of improving the development practices used previously and enhancing the user experience." data-locale-es="En este proyecto busque ofrecer como servicio el sistema que habia desarrollado en la version 1. Para lograrlo, tuve que mejorar las practicas de desarrollo que habia usado antes y elevar la experiencia de uso.">In this project, I aimed to offer the system I had developed in version 1 as a service. To achieve this, I faced the challenge of improving the development practices used previously and enhancing the user experience.</p>

<p data-locale-en="The main tasks I had to undertake included refactoring the frontend and improving the REST API standards for the backend." data-locale-es="Las tareas principales incluyeron refactorizar el frontend y mejorar los estandares REST de la API del backend.">The main tasks I had to undertake included refactoring the frontend and improving the REST API standards for the backend.</p>

<p data-locale-en="The goal of the frontend refactoring was to implement React.js for the booking calendar. In version 1, the calendar was built using HTML, CSS, and vanilla JavaScript. After migrating to the new technology, I reduced the amount of code by approximately 75%." data-locale-es="La refactorizacion del frontend tuvo como objetivo implementar React.js en el calendario de reservas. En la version 1, el calendario estaba construido con HTML, CSS y JavaScript vanilla. Despues de migrar a la nueva tecnologia, reduje aproximadamente un 75% la cantidad de codigo.">The goal of the frontend refactoring was to implement React.js for the booking calendar. In version 1, the calendar was built using HTML, CSS, and vanilla JavaScript. After migrating to the new technology, I reduced the amount of code by approximately 75%.</p>

<p data-locale-en="By adopting React.js, I achieved a direct improvement in performance for the websites using the calendar. Additionally, as an extra benefit, I now have more maintainable code that can also be used as an NPM library in other projects." data-locale-es="Al adoptar React.js, logre una mejora directa en el rendimiento de los sitios que usan el calendario. Ademas, como beneficio extra, ahora tengo un codigo mas mantenible que tambien puede reutilizarse como libreria NPM en otros proyectos.">By adopting React.js, I achieved a direct improvement in performance for the websites using the calendar. Additionally, as an extra benefit, I now have more maintainable code that can also be used as an NPM library in other projects.</p>

<p data-locale-en="On the backend side, improving REST practices enhanced the quality of data handling. I also implemented security improvements in the process of retrieving available time slots for different barbershops." data-locale-es="Del lado del backend, mejorar las practicas REST elevo la calidad del manejo de datos. Tambien implemente mejoras de seguridad en el proceso de obtencion de horarios disponibles para distintas barberias.">On the backend side, improving REST practices enhanced the quality of data handling. I also implemented security improvements in the process of retrieving available time slots for different barbershops.</p>

<p data-locale-en="When deploying the application, one of my tasks was managing the backend server, which runs on a VPS. As an example, I also deployed a frontend on a traditional hosting service." data-locale-es="Durante el despliegue de la aplicacion, una de mis tareas fue administrar el servidor backend, que corre sobre una VPS. Como ejemplo adicional, tambien desplegue un frontend en un hosting tradicional.">When deploying the application, one of my tasks was managing the backend server, which runs on a VPS. As an example, I also deployed a frontend on a traditional hosting service.</p>
