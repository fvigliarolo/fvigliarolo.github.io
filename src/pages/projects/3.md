<h2 data-locale-en="Walkthrough" data-locale-es="Recorrido">Walkthrough</h2>

<p data-locale-en="In this university assignment, along with two classmates, we faced the challenge of creating a system that used a NoSQL database service of our choice. As a team, we selected Firebase." data-locale-es="En esta entrega universitaria, junto con dos companeras, encaramos el desafio de crear un sistema que utilizara un servicio de base de datos NoSQL a eleccion. Como equipo, elegimos Firebase.">In this university assignment, along with two classmates, we faced the challenge of creating a system that used a NoSQL database service of our choice. As a team, we selected Firebase.</p>

<p data-locale-en="The task consisted of developing a backend with any technology we wanted, as long as it performed NoSQL queries. Additionally, the final result had to be delivered in Docker and include automated tests triggered by an orchestrator." data-locale-es="La consigna consistia en desarrollar un backend con la tecnologia que quisieramos, siempre que realizara consultas NoSQL. Ademas, el resultado final debia entregarse en Docker e incluir pruebas automatizadas ejecutadas por un orquestador.">The task consisted of developing a backend with any technology we wanted, as long as it performed NoSQL queries. Additionally, the final result had to be delivered in Docker and include automated tests triggered by an orchestrator.</p>

<p data-locale-en="For the automated tests, I decided to create Postman scripts for backend requests. After defining the test cases, I structured a command using Newman, Postman's command-line tool." data-locale-es="Para las pruebas automatizadas, decidi crear scripts de Postman para las requests del backend. After defining the test cases, I structured a command using Newman, Postman's command-line tool.</p>

<p data-locale-en="Finally, we used Jenkins to automate the execution and generate reports of the tests run with Newman." data-locale-es="Finalmente, usamos Jenkins para automatizar la ejecucion y generar reportes de las pruebas corridas con Newman.">Finally, we used Jenkins to automate the execution and generate reports of the tests run with Newman.</p>
